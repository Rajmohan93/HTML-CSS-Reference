# Learnings 

## Text in center of DIV

If we have only text inside the DIV, please prefer the approach 1

### Approach 1
```html
<div class="col text-center">
   <div>some content</div>
</div>
```

### Approach 2

```html
<div class="d-flex justify-content-center">
   <div>some content</div>
</div>
```
